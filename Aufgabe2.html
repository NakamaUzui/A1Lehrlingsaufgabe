Wie kommt eine Website auf den Bildschirm?:

Wie funktioniert das alles jetzt? Ich versuche, von meinem Computer auf die Website www.a1.net zuzugreifen. Mein Computer verwendet meine IP-Adresse, um Anfragen an das Modem zu senden. Falls der Modem die Seite nicht erkennt, leitet das Modem die Anfrage zusammen mit der IP-Adresse an den Provider (Internet Service Provider ISP) weiter. Der Provider sendet nun Anfragen an das globale Domain Name System (DNS). DNS ermittelt, welche IP-Adressen zu der gesuchten Domain gehören. Wenn jemand direkt vor mir google.com anrief, würde sich der DNS-Server das merken und die richtige IP-Adresse zurückgeben.[Zum Bild] Anhand der IP-Adresse weiss der Router dann, an welchen anderen Router er meine Anfrage schicken soll und landet schliesslich bei einem Server mit den angeforderten Information (in meinem Beispiel der Server von a1.net).
